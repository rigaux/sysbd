  • Dire qu’une requête est déclarative, c’est dire que (indiquer les phrases correctes) :

    1. La requête ne définit pas précisément le résultat.

    2. La requête ne dit pas comment calculer le résultat.

    3. La requête est indépendante de l’organisation des données.

    4. La requête est une expression de besoin en langage naturel.

  • Un plan d’exécution, c’est

    1. Un programme choisi parmi un ensemble fixe et pré-défini de programmes proposés par le système.

    2. Un programme produit à la volée par le système pour chaque requête.

    3. Un arbre d’opérateurs communicants entre eux.

  • L’optimisation de requêtes, c’est

    1. Modifier une requête SQL pour qu’elle soit la plus efficace possible.

    2. Structurer les données pour qu’elles soient adaptées aux requêtes soumises.

    3. Choisir, pour chaque requête, la meilleure manière de l’exécuter.

  • Quelles sont les affirmations vraies parmi les suivantes,? (1 pt)

  1. Le choix d’un plan d’exécution dépend de la mémoire RAM disponible.

  2. Le choix d’un plan d’exécution dépend de la forme de la requête SQL

  3. Le choix d’un plan d’exécution dépend de l’existence d’index

  4. Le choix d’un plan d’exécution dépend du langage de programmation utilisé.
